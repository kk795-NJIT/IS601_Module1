## 01) ls — List files and directories
**What it does:** Lists directory contents.

**Common flags:**
- `-l` → long format (permissions, size, dates)
- `-a` → include hidden files
- `-h` → human-readable sizes

**Examples:**
```bash
ls -la
ls -lh /Applications
